Web application for testing

Приложение, выбранное в качестве темы для курсового проекта. Выполнено на языке программирования Python с использованием фреймворка Django. В качестве базы данных выступила SQLite3, которая по умолчанию поддерживается как Python, так и Django.

Почему Python?

Python - это высокоуровневый, набирающий высокую популярность язык программирования, новые версии которого стабильно выходят раз в месяц-два. Этот язык программирования официально не подвергался никакой стандартизации, однако благодаря стандартам PEP8 и принципам SOLID программировать на Python стало на самом деле легко. В 2018 году Python занял первое место в рейтинге TIOBE среди самых популярных языков программирования.

Коротко о Django

Django - это высокоуровневый веб-фреймворк с открытым исходным кодом, который обеспечивает быструю разработку, а так же чистый и прагматичный дизайн. Django позволяет разработчикам сосредоточиться именно на написании чистого и красивого кода, а не на "изобретении колеса".
